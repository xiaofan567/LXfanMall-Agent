# LXfanMall 购物指南

## 一、平台简介

LXfanMall（乐享范商城）是一家综合性网上购物平台，主营品类涵盖：
- **服装鞋帽**：外套、T恤、休闲裤、牛仔裤、衬衫、男鞋
- **手机数码**：手机通讯、手机配件、摄影摄像、影音娱乐、数码配件、智能设备
- **家用电器**：电视、空调、洗衣机、冰箱、厨卫大电、厨房小电、生活电器、个护健康
- **家具家装**：厨房卫浴、灯饰照明、五金工具、卧室家具、客厅家具
- **汽车用品**：全新整车、车载电器、维修保养、汽车装饰
- **电脑办公**：平板电脑、笔记本、硬盘

合作品牌包括：Apple、华为、格力、方太、小米、OPPO、七匹狼、NIKE、海澜之家等知名品牌。

## 二、注册与登录

### 2.1 注册流程
1. 访问 LXfanMall 首页
2. 点击右上角「登录/注册」按钮
3. 切换到「注册」选项卡
4. 输入手机号码
5. 点击「获取验证码」并输入收到的验证码
6. 设置登录密码
7. 点击「注册」完成

### 2.2 登录方式
- 手机号 + 密码登录
- 手机号 + 验证码登录
- 登录凭证有效期 7 天

### 2.3 账户安全建议
- 设置高强度密码（字母 + 数字 + 特殊字符）
- 不要在公共设备上勾选「记住密码」
- 定期修改密码

## 四、商品详情

### 4.1 商品信息
在商品详情页（`/product/:id`）可以查看：
- **商品图片**：多角度展示图，支持放大查看
- **价格信息**：当前售价、原价、促销价、会员价
- **SKU 选择**：选择颜色、尺码等规格
- **商品描述**：详细的商品说明和参数
- **品牌信息**：品牌名称和介绍
- **销量数据**：累计销售数量
- **库存状态**：当前可购买数量
- **用户评价**：其他买家的评价和晒图

### 4.2 价格类型说明
| 价格类型 | 说明 |
|----------|------|
| 原价 | 商品标准售价 |
| 促销价 | 单品促销优惠价格 |
| 会员价 | 不同会员等级享受不同价格 |
| 阶梯价 | 购买数量越多，单价越低 |
| 满减价 | 满足金额条件后自动减免 |
| 秒杀价 | 限时限量特惠价格 |

## 五、购物车使用

### 5.1 添加商品到购物车
1. 在商品详情页选择好规格（颜色、尺码等）
2. 点击「加入购物车」按钮
3. 页面右上角购物车图标会显示商品数量

### 5.2 管理购物车
进入购物车页面（`/cart`）可以：
- 修改商品数量（增减或直接输入）
- 删除不需要的商品
- 选择需要结算的商品
- 查看商品小计和总计金额
- 查看优惠信息（促销优惠、优惠券优惠）

### 5.3 结算流程
1. 在购物车中勾选要结算的商品
2. 点击「去结算」按钮
3. 进入订单确认页面

## 六、下单与支付

### 6.1 订单确认页
在订单确认页（`/order`）需要确认：
- **收货地址**：选择或新增收货地址
- **商品清单**：确认购买的商品和数量
- **配送方式**：查看运费信息
- **优惠信息**：
  - 选择可用优惠券
  - 使用积分抵扣（100 积分 = 1 元，最多抵扣 50%）
- **订单金额**：商品总额 - 优惠金额 + 运费 = 应付金额

### 6.2 提交订单
确认信息无误后点击「提交订单」，系统将：
1. 验证商品库存
2. 计算优惠金额
3. 锁定库存
4. 生成订单号
5. 跳转到支付页面

### 6.3 完成支付
- 目前支持**支付宝**支付
- 支持 PC 网页支付和手机网页支付
- 支付完成后页面自动跳转到订单详情

### 6.4 支付时限
- 普通订单：120 分钟内完成支付
- 秒杀订单：60 分钟内完成支付
- 超时未支付订单将自动取消

## 七、订单管理

### 7.1 查看订单
在「我的订单」页面（`/orders`）可以按状态筛选：
- **全部订单**：所有订单记录
- **待付款**：等待支付的订单
- **待发货**：已付款等待发货
- **已发货**：商品已发出
- **已完成**：交易完成
- **已关闭**：已取消的订单
- **已退货**：退货流程中的订单
- **已评价**：已完成评价的订单

### 7.2 订单操作
| 订单状态 | 可用操作 |
|----------|---------|
| 待付款 | 去支付、取消订单 |
| 待发货 | 联系客服 |
| 已发货 | 查看物流、确认收货、申请退货 |
| 已完成 | 评价、申请退货、删除订单 |
| 已关闭 | 删除订单 |

### 7.3 物流查询
已发货订单可以查看物流轨迹，包含 11 个节点：
1. 包裹出库
2. 包裹出库，等待揽收
3. 快递员已揽收
4. 到达始发城市网点
5. 离开始发城市网点
6. 到达始发分拨中心
7. 离开始发分拨中心，发往目的城市
8. 到达目的分拨中心
9. 离开目的分拨中心，发往当地网点
10. 派送中，预计今日送达
11. 已签收

## 八、售后服务

### 8.1 申请退货
1. 在「我的订单」中找到对应订单
2. 点击「申请退货」
3. 填写退货原因和说明
4. 上传凭证图片（可选）
5. 提交申请

### 8.2 退货原因
可选的退货原因包括：
- 商品质量问题
- 商品与描述不符
- 商品破损/配件缺失
- 尺码不合适
- 颜色/款式不想要了
- 收到商品与订单不符
- 七天无理由退货
- 其他

### 8.3 退货流程
1. 提交退货申请 → 状态：待处理
2. 商家审核通过 → 状态：退货中
3. 退款完成 → 状态：已完成

如商家拒绝退货，状态变为「已拒绝」，可联系客服申诉。
